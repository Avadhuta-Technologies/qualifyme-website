# QualifyMe Website Structure

## Goal
Design a multi-path website that serves:
- Recruiters / Hiring Teams (primary buyers)
- Technical Evaluators (CTOs, engineering leaders)
- Investors (due diligence)
- Candidates (trust and experience)

# 11. IMPLEMENTATION NOTES

---

## UX Principles
- One idea per section
- High scan-ability
- Minimal text blocks
- Frequent CTAs

## Design Patterns
- Cards for roles/use cases
- Tables for comparisons
- Accordions for deep content
- Sticky navigation

## Performance
- Lazy load heavy visuals (simulation previews)
- Optimize for fast first paint
